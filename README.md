# 集创赛芯原杯工程项目

## 项目文件说明

```
.
├── build                   # 编译生成的一些文件 
│   └── ...
├── csrc                    # 测试用的c代码
│   ├── all.h
│   ├── config.h            # 配置文件
│   └── main.cpp
├── resource                # 资源
├── vsrc                    # verilog
│   ├── ...
└── Makefile               
└── README.md
```

## 教程

在命令行运行
```
make run #编译并运行测试程序
make wave #编译并显示波形
```
